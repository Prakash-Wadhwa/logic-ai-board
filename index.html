Statements & Not Statements

Statement: A declarative sentence that is either true (T) or false (F).

Not Statements: Sentences that do not have a fixed truth value.

  • Interrogative (Questions ?)
  • Exclamatory (Emotions !)
  • Commands / Requests
  • Open Sentences (e.g., "x + 2 = 5" or "He is a good boy")

Logical Connectives

Words used to connect mathematical statements together:

  • AND (∧): Conjunction. (Keywords: and, but, yet, still, moreover). True ONLY when both are T.
  • OR (∨): Disjunction. True if AT LEAST ONE is T. False ONLY when both are F.
  • IF-THEN (→): Implication / Conditional. False ONLY when T → F.
  • IFF (↔): Biconditional. True when both have the same truth value.

  • Remember: Words like 'Neither...nor' mean ~p ∧ ~q.
  • And watch out for 'but' - logically, it works exactly like 'AND' (∧)!

Finding Truth Values (Case 1)

Case 1: Given Truth Values.

If p, q are True (T) and r, s are False (F).

  • Evaluate: (p ∨ r) ∧ (q → s)

Step 1: Substitute values ⇒ (T ∨ F) ∧ (T → F)

Step 2: Solve brackets ⇒ (T) ∧ (F)

Step 3: Final Answer ⇒ F

Finding Truth Values (Case 2)

Case 2: Reverse Truth Values.

If p → (q ∨ r) is False. Find p, q, r.

  • Implication (→) is False only when T → F
  • Therefore, p ≡ T and (q ∨ r) ≡ F
  • OR (∨) is False only when both are F.
  • Therefore, q ≡ F and r ≡ F.

Quantifiers

Universal (∀): "For all".
The statement must be true for every single element in the set.

Existential (∃): "There exists".
The statement must be true for at least one element.

Duality

To find the dual of a statement, strictly swap the following:

  • ∧ ↔ ∨
  • t ↔ c (tautology / contradiction)
  • T ↔ F

Warning: Never change the negation (~) sign while taking duals!

Complete Guide to Negation

1. Verbal Form (Quantifiers):

  • "All" changes to "Some...not"
    e.g., "All boys are tall" → "Some boys are not tall"
  • "Some" changes to "No" or "None"
    e.g., "Some equations have roots" → "No equation has roots"

2. Symbolic Form:

  • ~(p ∧ q) ≡ ~p ∨ ~q
  • ~(p ∨ q) ≡ ~p ∧ ~q
  • Super Important:
    ~(p → q) ≡ p ∧ ~q
    (Notice: No 'if-then' in the negation!)
  • ~(p ↔ q) ≡ (p ∧ ~q) ∨ (q ∧ ~p)

Converse, Inverse, Contrapositive

Given an original conditional statement: p → q

  • Converse (q → p): Reverses the direction.
  • Inverse (~p → ~q): Negates both parts.
  • Contrapositive (~q → ~p): Reverses AND negates.

Prof. Wadhwa's Mirror Trick

Think of p → q as a mirror.
Converse: Swap places. | Inverse: Add 'Not' to both. | Contrapositive: Swap & Add 'Not'.

Board Exam Secret: A statement is ALWAYS equivalent to its Contrapositive!

Essential Laws of Logic

  • Conditional Law: p → q ≡ ~p ∨ q (Removes "if-then")
  • Idempotent Law: p ∨ p ≡ p  |  p ∧ p ≡ p
  • Distributive Law:
    p ∨ (q ∧ r) ≡ (p ∨ q) ∧ (p ∨ r)  |  p ∧ (q ∨ r) ≡ (p ∧ q) ∨ (p ∧ r)
  • Identity Law: p ∨ F ≡ p , p ∨ T ≡ T  |  p ∧ F ≡ F , p ∧ T ≡ p
  • Complement Law: p ∨ ~p ≡ T  |  p ∧ ~p ≡ F
  • Absorption Law: p ∨ (p ∧ q) ≡ p  |  p ∧ (p ∨ q) ≡ p

Truth Tables Check

When creating truth tables to prove equivalence or check patterns:

  • Tautology (t): The last column contains ONLY 'T'.
  • Contradiction (c): The last column contains ONLY 'F'.
  • Contingency: The last column contains a mixture of 'T' and 'F'.

Logical Equivalence Conclusion:
"Since columns (x) and (y) are identical, the given statements are logically equivalent."

Logic in Switching Circuits

Highest scoring part! Think of switches as statements.

Switch S₁ is closed = p

Switch S₂ is closed = q

Switch S'₁ is closed = ~p

Switches in SERIES (AND ∧)

Current flows only if S₁ AND S₂ are closed.

Symbolic form: p ∧ q

Complex Circuit Simplification

Symbolic form: (p ∧ q) ∨ (~p ∧ q)

Simplification Steps:

(p ∧ q) ∨ (~p ∧ q) ≡ (p ∨ ~p) ∧ q
[Distributive Law - factoring out q]

≡ T ∧ q
[Complement Law]

≡ q
[Identity Law]

Conclusion: The entire circuit is logically equivalent to ONLY switch S₂
